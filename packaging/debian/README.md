# Templates do pacote Debian

Esta árvore contém somente arquivos de integração e templates. O conteúdo de
`/opt/smartfile`, os ícones rasterizados e o arquivo `control` final são gerados
por `scripts/build_linux_deb.sh` em `build/linux/debian-root`.

Nenhum banco, token, documento ou configuração do desenvolvedor é copiado.
