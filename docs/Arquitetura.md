# Arquitetura

## Camadas

- Views: telas e componentes da interface
- Controllers: orquestração entre interface e lógica
- Services: regras de negócio
- Repositories: acesso a dados locais
- Models: representação de dados da aplicação

## Banco de dados

O banco local é mantido em SQLite e centraliza informações do Mini GED e histórico de uso.
