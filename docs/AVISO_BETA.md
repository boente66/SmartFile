# Aviso importante — versão de protótipo

Este pacote do SmartFile é um **protótipo beta para testes**. Ele ainda não é
uma versão oficial ou indicada para uso em produção.

Quem utilizar esta versão deve:

- manter cópias de segurança independentes dos documentos;
- não utilizar o SmartFile como única cópia de arquivos importantes;
- relatar erros, comportamentos inesperados e incompatibilidades encontrados;
- informar no relato a distribuição Linux, versão, arquitetura e etapas para
  reproduzir o problema;
- remover dados pessoais, documentos, tokens, Client IDs e segredos antes de
  anexar imagens ou logs.

Os relatos devem ser enviados pelo rastreador de issues do projeto no GitHub.
