# Manual do Usuário

Este documento é um placeholder inicial para orientar os primeiros passos com o SmartFile.

## Como usar

1. Execute o aplicativo com o comando `python run.py`
2. Importe documentos e organize-os localmente
3. Use as ferramentas de conversão, PDF e scanner conforme necessário
4. Consulte o Mini GED para buscar e revisar documentos
